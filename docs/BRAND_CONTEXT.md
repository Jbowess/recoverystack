## RecoveryStack Brand Context

This SEO system is not the main public website.

Primary brand/site:
- `https://recoverystack.io`

Primary conversion goal for this system:
- Drive organic SEO users to `https://recoverystack.io/news`
- Convert those users into health/tech newsletter subscribers

Downstream monetization goal:
- Sell the Volo Ring by RecoveryStack, a tech-enabled recovery ring
- Secondary path: send users to the main RecoveryStack site when appropriate

Funnel for this repo:
1. Capture high-intent organic traffic with SEO content
2. Move users to the RecoveryStack newsletter at `recoverystack.io/news`
3. Nurture via newsletter
4. Convert subscribers into Volo Ring customers or main-site visitors

Implications for product/SEO decisions:
- CTAs in this repo should prioritize the newsletter funnel over direct product purchase
- "Main site" links should point to `recoverystack.io`
- Product messaging should support newsletter conversion, not compete with it
- Recommendations should optimize for organic acquisition, email capture, and newsletter-to-product conversion
