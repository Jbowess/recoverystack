# Template: New Tech Spotlight (trends)
- Open with a concise intro of 60 words max.
- Summarize trend signal, credibility score, practical impact, and buyer risk.
- Build a mandatory verdict block with exactly 3 bullets in order:
  1) Best for: ...
  2) Avoid if: ...
  3) Bottom line: ...
- Include at least 3 FAQs.
- Mandatory CTA context: ring + $1/mo newsletter + free protocol PDF.
- Include a "What we don't know yet" section covering unresolved limitations or missing long-term data.
- All internal links MUST use markdown format: `[descriptive anchor text](/template/slug)`. Never use bare paths like `/guides/slug` without an anchor.
